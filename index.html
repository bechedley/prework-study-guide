HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags are used to structure and format data.
  • The data within a tag is known as an element.
  • Image tags must contain a src tag with a link to the image source.
  • Alt text is important for people using accessibility aids to access content, and it also improves SEO results.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS determines the visual elements of a webpage.
  • Visual elements dictated by CSS include colours and fonts.
  • CSS formating can be inline, internal or external via a linked .css file.
  • External CSS files located in the assets folder are preferrable.

Git

  • Git creates version histories of your work.
  • Correctly structuring your workflow enables easy recovery and debugging with git.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A adds updates all changes to the branch.
  • git commit -m"text" commits the changes to the branch, along with a descriptive message.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript controls interactive elements of a webpage.
  • This includes things like clicks and animations.
  • Functions must be called in order to work and always needs parenthesis.
